Installation
============

You can install the package using pip:

.. code:: shell

    pip3 install rex

Cite rex

If you are using rex for your scientific publications, please cite:

.. code:: bibtex

    @article{rex,
        author  = {van der Heijden, Bas and Luijkx, Jelle, and Ferranti, Laura and Kober, Jens and Babuska, Robert},
